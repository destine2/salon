# Veloura Beauty Studio

A conversion-focused luxury beauty salon website built for modern Nigerian beauty brands — WhatsApp booking, premium UX, pricing psychology, and mobile-first design.

**Brand:** Veloura Beauty Studio · Lekki Phase 1, Lagos

## Live preview

Open `index.html` in a browser, or serve the folder locally:

```bash
# Python
python -m http.server 8080

# Node (npx)
npx serve .
```

Then visit `http://localhost:8080`

## Pages

- `index.html` — Homepage (hero, services, pricing, gallery, team, testimonials, booking)
- `about.html` — About & team preview
- `service.html` — Services
- `contact.html` — Booking & visit
- `team.html` — Team
- `testimonial.html` — Client stories
- `404.html` — Not found

## Key features

- WhatsApp appointment booking (`js/booking.js`)
- Premium pricing & service CTAs
- Before/after transformation gallery with filters
- Owl Carousel hero & testimonials
- Google Maps contact block (`js/veloura-map.js`)
